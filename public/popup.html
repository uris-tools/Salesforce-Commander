Default shortcut keys
(customize your options at chrome://extensions/shortcuts)

  • Control + Shift + Space: Open Salesforce Commander
  • Control + Shift + A: Lightning App Menu
  • Control + Shift + 1: Tasks
  • Control + Shift + 2: Reports
  • [NEW FEATURE] Direct access to subpages for different objects like:
    • Access user pages directly by typing "users [partial name]"
    • Access custom object fields directly "[object name] fields Tab [field name]". For Example "account fields Tab name"

If you have any issues with stuff missing, type "refresh metadata" in the commander, hit enter, then refresh the page and try again.

Submit any issues on github :)
